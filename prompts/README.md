# 🎨 AI 프롬프트 보관함

이 폴더는 캠핑 허브 사이트와는 **별개**로, ChatGPT/Sora/Midjourney 등 AI 도구에 사용하는 프롬프트 텍스트를 보관하는 용도입니다.

## 사용 방법

1. 원하는 프롬프트 파일을 GitHub에서 열기
2. raw URL로 이동 → 텍스트 복사
3. AI 도구(ChatGPT 등)에 붙여넣기
